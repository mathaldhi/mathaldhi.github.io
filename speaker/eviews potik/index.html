Pendahuluan

Regresi linier berganda merupakan metode statistik yang digunakan untuk menganalisis hubungan antara satu variabel dependen dengan dua atau lebih variabel independen...

Bagian 1: Import Data ke EViews

Langkah-langkah untuk mengimpor data ke dalam EViews:

  • Buka EViews dan pilih File → Open → Foreign Data as Workfile.
  • Pilih file Excel (CSV/XLSX) yang sudah diunduh dari World Bank.
  • Pilih sheet yang berisi data dan klik OK.
  • Pastikan data sudah terbuka di jendela Workfile.

Bagian 2: Regresi Linier Berganda pada Data Time Series

Langkah-langkah untuk melakukan regresi linier berganda pada data time series:

  • Buka Workfile yang berisi data time series.
  • Klik Quick → Estimate Equation.
  • Masukkan model regresi, misalnya:
    • GDP c EduExp HealthExp Unemployment
  • Pilih metode Ordinary Least Squares (OLS) dan klik OK.
  • Hasil regresi akan muncul dalam output window.

Bagian 3: Regresi Linier Berganda pada Data Cross-Section

  • Buka Workfile yang berisi data cross-section.
  • Klik Quick → Estimate Equation.
  • Masukkan model regresi, misalnya:
    • HDI c DemocracyIndex GDPperCapita LiteracyRate
  • Pilih metode OLS dan klik OK.
  • Periksa output regresi dan interpretasi hasilnya.

Bagian 4: Regresi Linier Berganda pada Data Panel

  • Buka Workfile yang berisi data panel.
  • Pastikan struktur data sudah dalam bentuk panel.
  • Klik Quick → Estimate Equation.
  • Masukkan model regresi, misalnya:
    • GDP c EnvRegulation CO2Emission ForeignInvestment
  • Pilih metode **Pooled OLS, Fixed Effects (FE), atau Random Effects (RE)**.
  • Jika ingin memilih model terbaik, lakukan:
    • Uji Chow: View → Fixed Effects Tests.
    • Uji Hausman: View → Random Effects Tests.
  • Interpretasikan hasil regresi berdasarkan output.

Lampiran: Sumber Data dari World Bank

  • Pengeluaran Pendidikan (% PDB)
  • Pengeluaran Kesehatan (% PDB)
  • Tingkat Pengangguran
  • PDB per Kapita (US$ saat ini)
  • Tingkat Melek Huruf Dewasa (% usia 15 tahun ke atas)
  • Emisi CO2 (metrik ton per kapita)
  • Investasi Asing Langsung, arus masuk bersih (US$ saat ini)
