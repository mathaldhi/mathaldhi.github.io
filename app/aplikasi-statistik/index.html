      Klik Hitung jika data sudah benar.

      Perhitungan rata-rata secara manual.

      1. Menghitung banyaknya data. Contoh datanya adalah $a_1,a_2,a_3$ ada sebanyak $3$ data.
      2. Menjumlahkan semua data. Contoh $a_1+a_2+a_3=A$
      3. Membagi jumlah semua data dengan banyaknya data. Contoh $\mu=\overline{x}=\dfrac{A}{3}.$
      4. Rata-rata adalah $\mu$ atau $\overline{x}.$

    4. Data harus dipisahkan dengan salah satu dari koma (,) atau enter (pindah baris). Pemisahan desimal menggunakan titik (.) bukan koma (,). Hasil akan ditampilkan di sini.
      Klik Hitung jika data sudah benar.

      Perhitungan median secara manual.

      1. Menghitung banyaknya data. Contoh datanya adalah $1,4,3$ ada sebanyak $3$ data.
      2. Mengurutkan data. Contoh $1,3,4$
      3. Mencari tempat median dengan rumus $\dfrac{n+1}{2}$. Contoh $\dfrac{3+1}{2}=$$2.$
      4. Median adalah data ke-$2$ dalam contoh ini adalah $3.$

      Perhitungan jangkauan secara manual.

      1. Mencari data terbesar dan terkecil. Contoh $1,3,4,3$ maka data terbesar adalah $4$ dan data terkecil adalah $1.$
      2. Mengurangi data terbesar dengan data terkecil. Contoh ini adalah $4-1=3.$
      3. Jangkauan data adalah $3.$
